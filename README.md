## About

In this lab, you will practice the full Git and GitHub workflow for a small project. You will create a local folder, turn it into a Git repository, make commits, connect it to GitHub, push your work, and submit your GitHub repository link.

This workflow is important because developers use Git and GitHub every day to save work, share code, and show project history.
